# FTC 2025–2026 Robot

Control system for our FIRST Tech Challenge competition robot.

## Features
- TeleOp control
- Autonomous routines
- Intake system logic
- Drivetrain control

## Tech Stack
- Java
- FTC SDK
- Android Studio

## Structure
- TeamCode – custom robot logic
- FtcRobotController – SDK controller module

Developed for the 2025–2026 FTC season.
